## Projetos

### 🏭 [Controle de Produção CCL](https://github.com/carlosbill/controle-producao)

> Solução desenvolvida em C# para controle de quebra e perda de materiais no processo produtivo industrial.

Problema real que identifiquei no meu setor e resolvi com código.

**O que o sistema faz:**
- Recebe dados de impressão e revisão de cada ordem de produção
- Calcula automaticamente perda e porcentagem de quebra
- Retorna sinalizador de status: 🟢 Bom · 🟡 Ok · 🔴 Ruim
- Salva histórico em CSV para acompanhamento por turno e matrícula

**Conceitos aplicados:** Classes · Objetos · Encapsulamento · Métodos · Construtores · Coleções · Persistência em arquivo

```
Tecnologia: C# · Console Application · .NET
Status: ✅ Funcional
```
